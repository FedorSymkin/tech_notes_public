# Типы дорог
* Есть 4 типа дорог, и на них немного разные правила, потому что разные скорости:
    * В НП (город) - 60 км/ч по умолчанию
    * Вне НП (трасса) - 90 км/ч по умолчанию
    * Автомагистраль - 110 км/ч по умолчанию
    * Жилая зона - 20 км/ч по умолчанию, а пешеходам можно везде ходить
* Тип дороги определяется **только** знаками, а не застройкой
* Есть знак города на синем фоне - он не считается за начало и конец города, это просто для инфы
